# cnpm-server
## For Linux and Mac 
```
$ sh run.sh
```
## For Windows
### Install requirements
```
pip install -r requirements.txt
```
### Makemigrations
```
python manage.py makemigrations
```
### Migrate
```
python manage.py migrate
```
### Run
```
python manage.py runserver
```
